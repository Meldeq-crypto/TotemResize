
A simple, client-side Fabric mod that allows you to change the size and scale of the Totem of Undying. Perfect for PvP or survival players who find the vanilla totem model too distracting or bulky on their screen.

## ✨ Features
* **Custom Scaling:** Shrink or enlarge the totem model to your liking.
* **Offhand Fix:** Properly handles rendering so the totem looks right in both the main hand and offhand.
* **Client-Side Only:** No server-side installation required. Works on any server.
* **Lightweight:** Minimal impact on game performance.

## 🛠 Installation
1. Make sure you have **Fabric Loader** installed.
2. Download the latest release from [Modrinth](https://modrinth.com) or the Releases tab here.
3. Drop the `.jar` file into your `mods` folder.
4. (Optional) Install **Cloth Config** and **Mod Menu** to adjust settings in-game.

## ⚙️ Configuration
You can adjust the totem scale in the config file located at:
`.minecraft/config/totempop.json`

| Option | Default | Description |
| :--- | :--- | :--- |
| `Enabled` | true | Toggle the mod on/off |
| `Scale` | 0.4 | The size of the totem (1.0 is vanilla) |
| `X-Offset` | 0.0 | Adjust horizontal position |
